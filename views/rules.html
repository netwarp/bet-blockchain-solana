Rules

It's a mini-game on the Solana's blockchain.
The only network available for now is devnet.
The users send a mini-amount to the kitty (0.001 Sol)
A signature transaction is sent to the network.
The transaction is registered in a block.
If the latest signature's number is equal to the latest recent blockhash's number, the user win the kitty.
For example, let's see this transaction in a explorer:
